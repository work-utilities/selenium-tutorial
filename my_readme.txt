1. inspect screen element - e.g. google input 

xpath syntax:
//tagname[@attribute='value']

example - google search box:
//input[@name='q']

Note: 
You can ctrl + F > opens a search box in debug area and put there:
//input[@name='q']

This will highlight the input element which has name 'q'
